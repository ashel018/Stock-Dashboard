# 📈 Stock Price Prediction Dashboard

A full-stack web application that allows users to visualize stock prices, view live market data, and predict future prices using machine learning.

---

##  Features

*  Interactive stock price charts
*  Live stock price display
*  Price prediction using Linear Regression
*  Web-based dashboard using Flask
*  Real-time data using yfinance API

---

##  Tech Stack

* **Backend:** Python, Flask
* **Frontend:** HTML, CSS, JavaScript, Chart.js
* **Machine Learning:** Scikit-learn
* **Data Source:** yfinance API

---

##  Installation

```bash
pip install -r requirements.txt
python app.py
```

---

##  Usage

1. Enter a stock symbol (e.g., AAPL, TSLA, RELIANCE.NS)
2. Click "Get Data"
3. View:

   * Stock chart 
   * Live price 
   * Predicted price

##  Example Stocks
